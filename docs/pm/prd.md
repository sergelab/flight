# PRD: flight v0.2

## Цель
Сделать графическую программу "бесконечного" полёта над гористой поверхностью (генерация случайная, детерминированная по seed).

## Scope v0.2
Входит:
- Автополёт вперёд с постоянной скоростью, без поворотов
- Высота камеры следует рельефу + фиксированный оффсет (сглаженно)
- Бесконечная поверхность (чанки) без заметных фризов
- Запуск через Poetry: `poetry run flight`

Не входит:
- Управление самолётом/камерой (кроме выхода)
- Текстуры, объекты, вода, небо (кроме базовой заливки)
- Сложная эрозия/LOD

## Критерии приёмки
- Приложение запускается командой `poetry run flight`
- Камера летит вперёд непрерывно, мир подгружается бесконечно
- Нет заметных пауз при подгрузке чанков (генерация в фоне)
- Высота камеры визуально плавная (без резких скачков)

## Риски
- Особенности OpenGL на macOS (контекст и производительность)
- Фризы при создании GPU-ресурсов: минимизируются дозированной загрузкой чанков

## Метрики качества (ориентиры)
- FPS визуально плавный на типичном macOS ноутбуке
- Подгрузка чанков без стоп-кадров


## Дополнения v0.2
- Улучшение читаемости движения: контурные линии по высоте (шейдер)
- Тюнинг параметров мира через CLI
