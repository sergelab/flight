# PRD: flight v0.3 (Final)

## Цель
Сделать графическую программу "бесконечного" полёта над гористой поверхностью:
- запуск одной командой (`poetry run flight`)
- камера движется вперёд с постоянной скоростью
- высота камеры следует рельефу с фиксированным оффсетом и сглаживанием
- без поворотов камеры
- всё плавно (без дерганья, без мерцания картинки)

## Важное решение v0.3: Variant A (стабильность)
В v0.3 фиксируем визуальный baseline без shimmer:
- **без** процедурной "текстуры" поверхности (контуры/noise)
- только крупная форма рельефа + освещение + fog

Причина: на движении камеры тонкие procedural-детали дают мерцание на части GPU/драйверов (включая macOS).

## Scope v0.3
Входит:
- ModernGL + pygame окно
- генерация высот детерминированная по seed
- чанки мира, подгрузка по движению
- базовый шейдер (height palette + Lambert + forward fog)
- более разнообразная генерация ландшафта (domain-warp + macro слой)
- реки и озёра (визуализация цветом, без процедурных мелких паттернов)
- debug режим (`--debug`)

Не входит:
- “красивые” текстуры/материалы
- стабильный LOD как продуктовая фича (может быть экспериментально за флагом)
- управление камерой/повороты/ручной полёт

## Acceptance criteria
- `poetry run flight` запускается на macOS (Python 3.12)
- на экране виден рельеф, он движется под камерой (автополёт)
- картинка стабильная: нет заметного мерцания мелких деталей
- документация актуальна и соответствует коду
