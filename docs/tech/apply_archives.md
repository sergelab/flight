# Как применять архивы с правками

Мы поставляем изменения одним `.zip` архивом **с файлами под замену** (с сохранением структуры каталогов).

## Правильный способ (рекомендуется)
1) Откройте архив и убедитесь, что на верхнем уровне лежат папки/файлы вроде:
   - `flight/`
   - `docs/`
   - `README.md`
   - `pyproject.toml`
2) Распакуйте архив **в корень репозитория** с опцией **Replace/Overwrite**.

## Частая ошибка
Архив распакован в подпапку (например `flight-main/`), из-за чего файлы не заменились, а приложение продолжает запускать старый код.
Проверка: файл `flight/world/world.py` должен находиться по пути:
- `<repo_root>/flight/world/world.py`

## Проверка применённых изменений
Запустите:
```bash
poetry run flight --debug
```
и убедитесь, что в HUD отображается версия **v0.2** и новые параметры (chunk_res/fog).
