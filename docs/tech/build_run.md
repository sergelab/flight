# Сборка и запуск

## Требования
- Python 3.12
- Poetry
- OpenGL 3.2+ (желательно 3.3 core)

## Установка
```bash
poetry install
```

## Запуск
Ручной режим (по умолчанию, управление стрелками):
```bash
poetry run flight
```

Автополёт (как в предыдущих версиях):
```bash
poetry run flight --auto
```

## Управление
- ↑ — вперёд
- ↓ — назад
- ← — поворот влево
- → — поворот вправо
- Esc — выход

## CLI параметры (актуально для v0.8.0)

### Мир / генерация
- `--seed <int|random>` — сид
- `--noise fast|simplex` — режим шума высот

### Полёт / камера
- `--speed <float>` — скорость движения (world units/sec)
- `--height-offset <float>` — высота камеры над поверхностью
- `--auto` — автополёт вперёд (игнорирует стрелки вверх/вниз)
- `--turn-rate <float>` — скорость поворота в ручном режиме (рад/сек)

### Рендер
- `--wireframe` — каркас
- `--fog-start <float>` / `--fog-end <float>` — туман

### Чанки
- `--chunk-res <int>` — разрешение чанка (вершин на сторону)
- `--chunk-size <float>` — размер чанка в world units
- `--lod` / `--no-lod` — LOD кольца (по умолчанию выключено)

### Леса
- `--trees` / `--no-trees` — леса (по умолчанию включено)
- `--tree-density <float>` — множитель плотности

### Отладка / производительность
- `--debug` — HUD + логи
- `--target-fps <int>` — цель FPS для адаптивной подгрузки

## Замечания для macOS
- pygame использует системный OpenGL. При проблемах с запуском обновите pygame и убедитесь, что запускаете из GUI-сессии (не из headless окружения).
