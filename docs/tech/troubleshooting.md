# Troubleshooting

## Окно не открывается / чёрный экран
- Убедитесь, что запускаете в GUI окружении.
- Обновите драйверы/ОС (актуально для Windows/Linux).
- На macOS убедитесь, что pygame установлен корректно: `poetry show pygame`.

## Низкий FPS
- Уменьшите `CHUNK_RES` в `flight/config.py`
- Уменьшите окно (параметры в `flight/config.py`)

## Ошибка ModernGL: "Requested OpenGL version 330, got version 0"
Это означает, что OpenGL контекст не был создан/не стал текущим.

Что сделать:
- Убедитесь, что запускаете из обычной GUI-сессии (не headless/SSH).
- Обновите зависимости: `poetry update pygame moderngl`.
- На macOS проект запрашивает Core Profile 3.3; если у вас старая система, нужен минимум OpenGL 3.2.

## Горы не видны (только голубой фон)
Причина может быть в порядке обхода вершин (winding) и отсечении граней (culling).
В v0.1 включён безопасный режим: culling отключён, а индексы построены CCW.
